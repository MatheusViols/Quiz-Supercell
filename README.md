<h1> Quiz Supercell </h1>
Esse projeto foi solicitado para uma avaliação da segunda unidade da matéria Desenvolvimento Web no curso de Análise e Desenvolvimento de Sistemas.
O quiz apresenta 10 perguntas e 5 alternativas cada. As perguntas são todas relacionadas à empresa ou seu jogos.
A pontuação tem um sistema de recompensa por coroas do jogo Clash Royale, ao acertar uma pergunta você ganha uma coroa, ao acertar metade das perguntas uma segunda coroa e ao acertar todas ganha a terceira coroa.

O projeto não usa nenhuma conexão além de arquivos locais. 
Foi desenvolvido e testado utilizando os navegadores Chrome e Firefox.

Para iniciar apenas execute o index.html no navegador de preferência.
